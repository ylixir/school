if $x>1$ then $x^2>x$. If $0<x<1$ then $x^2<x$

\subsubsection*{proof}
\begin{align*}
  x^2&=x\cdot x\\
  x\cdot x&>0\text{ by axiom 8}\\
  x\cdot x&=x_1+x_2+...+x_x\\
  0&<x\\
  0+x&<x+x \text{ by axiom 7}\\
  x&<x_1+x_2+...+x_i \text{ by induction}\\
\end{align*}
Therefore $x<x^2$
$\Box$
